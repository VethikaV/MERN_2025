Git and Github

What is git?

It is a tool for controlling files and folders through commands

what is github?

Reprository of files stored at cloud

To check whether git si present or not?

git --version [in your command prompt]

command to link the local file to the git reprository?

git init [in comand prompt]

what is the command to know the status of the git?

git status[command prompt]

What is the command to add files in the git??

git add "filename"

Command to commit in git??

git commit -m "message"

Command to push a file to github?

git push -u origin "branch name"

How to bring back the changes done in global to local??

git pull[in command prompt]
